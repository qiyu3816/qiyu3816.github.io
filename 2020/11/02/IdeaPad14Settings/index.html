IdeaPad14新电脑Ubuntu配置过程中的问题（个人笔记）

​ 买新电脑后安装Ubuntu双系统，开始一系列基本配置。

​ 包括：

  • 安装opencv， sudo apt-get install libopencv-dev

  • 安装boost， sudo apt-get install libboost-dev

  • 安装cmake，直接apt-get

  • 安装vim并编辑配置文件，直接apt-get，~/.vimrc配置文件已备份到移动硬盘中
  • 配置全局默认consolas字体，
  • 安装zsh，后安装ohmyzsh，前者apt-get，后者用官网的命令连接失败，直接上github官方仓库链接下载，然后 sudo ./tools/install.sh 即可安装
  • zsh主题配置，我选择agnoster主题，在~/.zshrc配置文件中找到 ZSH_THEME 项更改即可
  • 安装tmux并配置个性化，直接apt-get，然后进~/.tmux.conf编辑，已备份到移动硬盘
  • 更改静态ip，先编辑好/etc/network/interfaces文件，因为没有网线接口，用转换器接，每次接都须重新 sudo /etc/init/d/networkings restart
  • 解决中途WiFi列表no network的问题，这里参照博客和其中的GitHub网址安装了rtw88驱动，成功解决，文件已备份到移动硬盘中
  • 解决触控板失灵问题（已失败），无论是安装驱动还是重载模块，全都没用
  • 安装ROS，没有自己瞎搞，全程参照Wiki教程
  • 安装vscode，照官网命令行安装
  • 安装typora，照官网命令行安装

End
